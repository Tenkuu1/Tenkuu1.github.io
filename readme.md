# How to 

## Prerequiresite

- Install visual code: https://code.visualstudio.com
- Install git: https://git-scm.com/download/win

## Install

- `git clone https://github.com/Tenkuu1/fuyuko.git`

## Check updates

- `git status`
- `git diff`

## Save new updates

- `git add .`
- `git commit -m "..."`
- `git push`
